## FP300Service, FP300 cihazlarıyla entegrasyon sağlamak amacıyla geliştirilmiş WPF NetCore tabanlı bir çözümdür. 

### Bu çözüm, Microsoft Visual Studio 2026 kullanılarak geliştirilmiştir.

### Versiyona ait DLL ler etc klasoru altinda bulunmaktadir.

### T300 entegrasyonu testlerinde derleme hatası almamak için, destek ekibimizden alacağınız "Certificates" klasörünü "fpu\CSharp\NetCore\FP300Service" konumuna atmanız gerekmektedir.
